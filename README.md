# eurobot-2026-pami

[![PlatformIO CI](https://github.com/Game-Team-SLSC/eurobot-2026-pami/actions/workflows/pio.yml/badge.svg)](https://github.com/Game-Team-SLSC/eurobot-2026-pami/actions/workflows/pio.yml)

This repo contains the source code for the standard PAMIs of Game Team 2026

# Usage

- Install [platformio CLI/IDE](https://platformio.org/)
- Clone the repo
- Plug the PAMI in
- Upload the code using the IDE or `pio run -t upload`

# Testing

- Ensure the PAMI battery is charged before use.
- Ensure heatsinks are correctly put on voltage regulators/motor drivers
